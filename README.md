# Taehyun Kim Portfolio — Chaeri-inspired layout

This version recreates the overall visual direction shown in the reference screenshots:

- ivory academic-style background
- large serif headings
- profile block at the top
- mobile side drawer menu
- News, Selected Work, Projects, Writing, and Contact
- English / Simplified Chinese switch
- light / dark mode

## Replace photos

- `assets/profile-placeholder.svg`
- `assets/hero-placeholder.svg`

Replace them with JPG or PNG files and update the filenames in `index.html`.

## Publish

Upload the entire folder to a GitHub repository and enable GitHub Pages from Settings → Pages.
